# Baseline Deep Learning

Eksperimen Prompt 2 memakai `MLPClassifier` sebagai baseline neural ringan.

- Jalur ini dipisahkan dari production dan tidak dipakai oleh app default.
- Pendekatan ini belum sequence-aware seperti LSTM atau transformer.
- Tujuannya memberi baseline neural yang murah untuk dijalankan lokal sebelum migrasi ke model yang lebih berat.

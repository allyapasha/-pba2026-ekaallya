# Documentation

Folder `docs/` berisi dokumentasi yang masih relevan untuk operasional dan evaluasi repository ini.

## Files

- `dataset_audit.md`
  Ringkasan dataset, mapping label, distribusi kelas, dan risiko data.
- `classic_vs_deep_learning.md`
  Perbandingan jalur production klasik dengan baseline eksperimen.
- `deployment_hf_space.md`
  Panduan deploy folder `apps/hf_space/` ke Hugging Face Space.

## Scope

Dokumen proses kerja sementara, handoff internal, dan prompt kerja tidak disimpan di folder ini agar repository tetap bersih untuk kebutuhan production dan presentasi.
